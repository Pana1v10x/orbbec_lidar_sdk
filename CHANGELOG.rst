Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

v1.0.4 (2025-03-28)
====================
### Added
- Lidar scan smoothing support
- Playback time display
- DNS query sending functionality
- Real-time motor speed retrieval
- Log message callback in scan viewer
- Enum interface support on Windows

### Changed
- Project version
- Example and README

### Fixed
- Compile error on Windows
- Recorder bug and CMake RPATH
- Missing headers in playback
- Product name, subnet mask, and MAC address retrieval
- Spin speed, scan speed, and mode settings
- Heartbeat mechanism
- Logging output
